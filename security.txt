# Security Policy for Analos Programs

## Reporting Security Vulnerabilities

If you discover a security vulnerability in any of the Analos programs, please report it responsibly:

1. **Email**: support@launchonlos.fun
2. **Twitter**: @EWildn
3. **Telegram**: t.me/Dubie_420

## What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Time

We aim to respond to security reports within 24 hours and provide updates within 72 hours.

## Scope

This security policy covers all programs in the Analos NFT Launchpad ecosystem:
- Price Oracle
- Rarity Oracle  
- NFT Launchpad
- Token Launch
- Enhanced Programs

## Rewards

We may offer rewards for valid security vulnerabilities depending on severity and impact.

## Disclosure

We will coordinate disclosure of vulnerabilities with reporters to ensure proper patching before public disclosure.
